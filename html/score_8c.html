◆ score_handle()

uint8_t score_handle ( )

Function to handle the score.

Score is incremented by one every second, and is sent to Node 1 each increment. It also check for game over, described by an IR led signal. On game over, a CAN signal with id = 5, length = 1 and its value being the final score is sent to Node 1.

Returns
0 if it has detected a game over, 1 otherwise.
